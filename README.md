# FinalProject 個人記帳系統(Personal Accounting System)

一、 專案簡介 
本專案開發了一款基於 C# Windows Forms 的個人記帳桌面應用程式。系統旨在幫助使用者輕鬆記
錄日常收入與支出，具備完整的資料增刪改查 (CRUD) 功能、關鍵字搜尋，以及資料永久儲存 (檔
案 I/O) 能力。此外，更加入了跨視窗的動態圓餅圖表分析功能，提供直覺的視覺化財務檢視。 
二、 使用說明書與功能展示 
1. 記帳資料管理 (新增 / 修改 / 刪除) 
使用者可於左側輸入區設定日期、收支類型、分類、金額與備註。 
系統內建「防呆機制」，若未選擇分類或輸入非數字金額，系統會跳出警告並阻擋錯誤資料寫入。 
點選右側表格 (ListView) 內的任一筆資料，系統會自動將資料帶回左側輸入框，方便使用者進行
「修改」或「刪除」。 
表格會依據收支類型自動套用顏色（支出為紅色、收入為綠色），提升閱讀性。

3. 關鍵字搜尋功能 
操作說明： 
於左下方的「搜尋」框內輸入關鍵字（例如：早餐、特定日期或備註）。 
點擊搜尋按鈕後，右側表格中符合條件的帳目，其背景會自動標示為「黃色螢光筆」效果。點選
「清除搜尋」即可恢復原狀。

5. 檔案存檔與讀取 (File I/O) 
操作說明： 
點擊「存入檔案」，系統會利用 StreamWriter 將當前所有帳目匯出並儲存至本機 CSV 檔案。 
關閉程式後重新開啟，點擊「讀取檔案」，系統會透過 StreamReader 將先前的資料完整還原至表格
中，達成資料的持久化保存。 



6. 財務統計儀表板與視覺化圖表 
操作說明： 
點擊「查看統計」按鈕，會彈出全新的統計分析視窗。 
畫面右側會依據實際金額比例，動態繪製出「甜甜圈圓餅圖 (Doughnut Chart)」。滑鼠游標停留在色
塊上時，會顯示精確金額。 
點擊左上角「匯出圖表」，可將此圓餅圖儲存為 PNG 圖片檔至電腦中。

三、 專案亮點與技術總結 
跨視窗資料傳遞：利用物件導向的建構子 (Constructor) 傳遞參數，成功將主視窗的總計數據傳遞至
統計視窗 (Form2/Total)。 
響應式 UI 設計：善用 Anchor 與 MinimumSize 屬性，確保視窗在放大或縮小皆不會跑版變形。 
防呆與錯誤攔截：大量使用 int.TryParse、string.IsNullOrWhiteSpace 及 SelectedIndex 進行輸入檢
查，提升系統穩健度。
